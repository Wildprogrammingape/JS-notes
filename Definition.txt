Function有两种定义方式

1) Function declaration (过时了不常用)

function functionName(parameters) {
  // code to be executed
}

如:

function myFunction(a, b) {
  return a * b;
}


2) Function Expression (也就是匿名函数anonymous function, 不需要function关键词)

如 :

var x = function (a, b) {return a * b};
var z = x(4, 3);


3) Arrow Function 箭头函数让function expression更简洁了, 不用function return 和 {}

例子:

// ES5
var x = function(x, y) {
  return x * y;
}

// ES6
const x = (x, y) => x * y;


4) self-invoking function自调用函数, 不需要call直接执行

语法类似于: Function expressions +  ().

例子:

<script>
(function () {
  document.getElementById("demo").innerHTML = "Hello! I called myself";
})();
</script>

-- Hello! I called myself
