jQuery相关的CSS style功能对比

1) Hiding HTML Elements 隐藏HTML元素

jQuery:

myElement.hide();

JavaScript:

myElement.style.display = "none";


2) Showing HTML Elements 显示HTML元素

jQuery:

myElement.show();

JavaScript:

myElement.style.display = "";

3) Styling HTML Elements 改变HTML元素字体大小

jQuery:

myElement.css("font-size","35px");

JavaScript:

myElement.style.fontSize = "35px";
