Event是HTML element变化的意思，

something changed on HTML element，比如用户点击按钮

语法：

<element event="some JavaScript">

例子：

<button onclick="document.getElementById('demo').innerHTML = Date()">The time is?</button>


常用的HTML event:

Event		Description

onchange		An HTML element has been changed			HTML element改变
onclick		The user clicks an HTML element			用户点击	
onmouseover	The user moves the mouse over an HTML element	   	鼠标悬停
onmouseout	The user moves the mouse away from an HTML element      鼠标离开
onkeydown	The user pushes a keyboard key 			按键盘
onload		The browser has finished loading the page		浏览器完成加载
