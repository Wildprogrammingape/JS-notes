BOM 是 Browser Object Model,  allows JavaScript to "talk to" the browser 用来让JS和浏览器交互

1) window

window对象是浏览器，因此有最高等级  

document.getElementById 和 window.document.getElementById 是一个意思

2) window size 

可以使用两个属性来确定浏览器窗口的大小

window.innerHeight -浏览器窗口的内部高度（以像素为单位）
window.innerWidth -浏览器窗口的内部宽度（以像素为单位）

其他一些方法：

window.open() -打开一个新窗口
window.close() -关闭当前窗口
window.moveTo() -移动当前窗口
window.resizeTo() -调整当前窗口的大小
